# AI Shot Generator

**描述**: 调用 ZenStudio zencli 执行镜头生成（图片+视频），执行前需通过 shot-config-builder 构建完整配置。

**职责范围**:
- ✅ 执行图片生成（zencli generate image）
- ✅ 执行视频生成（zencli generate video）
- ✅ 下载生成结果并整理到 output 目录
- ✅ 版本管理（同一镜头多次生成自动编号 v1/v2/v3...）
- ❌ 不负责 Prompt 构建（由 shot-config-builder 负责）

**触发条件**: 
- 用户说"生成镜头 XXX"
- 用户说"执行生成"
- shot_configs/ 下已有该镜头的配置文件

---

## 📐 标准化工作流（重要）

生成任何一个镜头，必须严格遵循以下顺序：

### Step 1: 解析香盘表 → 获取 Prompt
从 `~/Desktop/资产/香盘表/20250904镖人2-01集香盘表.xlsx` 读取对应镜号的「讲戏」字段作为基础 Prompt。

**示例**:
```
镜号 c003 → 讲戏字段 → "36长江水战"
```

### Step 2: 准备三类输入资产

#### A. 人设图（必须）
- 来源：`~/Desktop/资产/人设定稿/<角色名>/`
- 规则：严格使用「清稿/色稿/final」版本，排除「RR/线稿/sketch」版本
- 优先：三视图 > 全身 > 半身 > 特写
- 标记：青年刀马 ≠ 刀马17岁（区分！）

#### B. 场景氛围图（必须）
- 来源：`~/Desktop/资产/氛围/氛围/`（18张）
- 规则：从香盘表的「场景」字段匹配，如"长江-隋军马船"→匹配「灭陈水战.jpg」

#### C. 分镜图构图参考（必须）
- 来源：`~/Desktop/镖人2项目/storyboard/shot_XXX/sketch.png`
- 规则：作为构图参考图输入给 AI，确保镜头构图与分镜一致

### Step 3: 构建 Prompt
将「讲戏」字段 + 二维动画风格 + 各参考图说明组合成完整 Prompt：

```
【二维动画风格】
[从讲戏字段解析的场景描述]
构图严格参考分镜图。
人物严格参考[角色名]人设图。
场景氛围严格参考[场景图名称]。

确保：
- 角色形象与人设图完全一致
- 场景氛围与参考图一致
- 构图与分镜图一致
- 二维动画风格，清晰分层
```

### Step 4: 执行生图
```bash
zencli generate image \
  --prompt "[完整Prompt]" \
  --model "316" \
  --resolution "1" \
  --aspect-ratio "16:9" \
  --count "1" \
  --input-images "[场景CDN],[人设CDN],[分镜CDN]" \
  --poll
```

**默认值（必须使用）**:
- 模型：316 贝宝pro
- 分辨率：2K（--resolution "1"）
- 比例：16:9

### Step 5: 执行生视频
```bash
zencli generate video \
  --prompt "[二维动画风格的动态描述]" \
  --model "402" \
  --mode "Text" \
  --duration "5" \
  --count "1" \
  --single-image-url "[生成图CDN]" \
  --poll
```

**默认值（必须使用）**:
- 模型：402 zen-03

## ⚠️ 常见错误

### 错误1: 选错角色版本
- 青年刀马 ≠ 刀马17岁（完全不同角色）
- 陈国武将 ≠ 陈国士兵（不同角色）
- 排除所有含 RR/线稿/sketch/draft 的文件

### 错误2: Prompt 过于模糊
- 不能只说"长江水战"
- 必须包含具体人物、场景、动作描述
- 必须加上"严格参考[资产]"

### 错误3: 分辨率用错
- 生图必须用 `--resolution "1"`（2K）
- 不能用默认值（1K）
