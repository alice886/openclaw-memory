# PDF 分镜图提取工具 v8

## 📦 工具说明

专为分镜脚本 PDF 设计的图片提取工具，能够：
- ✅ 按镜号（c001, c002...）精确提取分镜图
- ✅ 自动过滤页眉/页码图片
- ✅ 正确处理跨页延续的子分镜
- ✅ 基于子分镜编号（1/4, 2/4）精确匹配图片数量

## 🎯 核心特性

### 1. 智能识别
- **主镜号**: c001, c002, c003...
- **子分镜**: 1/4, 2/4, 3/4, 4/4
- **跨页延续**: 自动处理子分镜跨页的情况

### 2. 精确匹配规则
- 子分镜的**分母**决定该镜号的图片总数
  - 看到 `1/4` → 该镜号有 4 张图片
  - 看到 `1/1` → 该镜号有 1 张图片
  - 看到 `1/5` → 该镜号有 5 张图片

### 3. 自动过滤
- 过滤页眉图片 (Y坐标 < 50)
- 过滤页码图片
- 只保留真正的分镜图

### 4. 命名格式
- `c001_1.jpeg`
- `c001_2.jpeg`
- `c001_3.jpeg`
- `c001_4.jpeg`

## 🔧 工作原理

### 1. 文本识别
- 提取 PDF 中所有文字块及其坐标
- 识别主镜号（c001-c999）
- 识别子分镜（1/4, 2/4 等）

### 2. 跨页处理
```
第2页:
  c001
  1/4, 2/4, 3/4, 4/4
  c002
  1/2  ← 第一张

第3页:
  2/2  ← 延续 c002 的第二张
  c003
  1/5, 2/5, 3/5, 4/5

第4页:
  5/5  ← 延续 c003 的第五张
  c004
  1/1
```

### 3. 图片匹配
- 每个子分镜标记匹配其右侧的图片（Y坐标容差 30px）
- 跨页子分镜继承上一页的主镜号
- 过滤页眉区域 (Y < 50) 的图片

### 4. 保存组织
- 按镜号创建文件夹
- 图片按页面和Y坐标排序
- 使用镜号前缀命名：`c001_1.jpeg`

## ⚙️ 高级配置

### 调整容差

如果需要调整匹配容差，可以修改脚本中的参数：

```python
# 图片与子分镜的Y坐标容差（默认 30 像素）
if distance < 30 and img_y >= sub_y - 20:

# 页眉过滤阈值（默认 50 像素）
if img_y < 50:
    filtered_count += 1
    continue
```

### 支持的PDF格式

- PDF 版本：1.4+
- 文字层：需要包含文字（不支持纯扫描版）
- 图片格式：JPEG, PNG 等常见格式

## 🐛 故障排查

### 问题：图片数量不对

**检查子分镜编号**：
```bash
# 运行脚本后会显示期望数量
python3 extract_v8.py your.pdf
```

**验证逻辑**：
- 子分镜的分母 = 该镜号应有的图片总数
- 如果不匹配，说明 PDF 中的子分镜标记有误

### 问题：缺少依赖

```bash
# 安装 PyMuPDF
pip3 install PyMuPDF

# 如果提示权限错误
pip3 install --user PyMuPDF
```

### 问题：提取速度慢

- 正常情况：263页 PDF 约需 2-3 分钟
- 如果超过 10 分钟，检查 PDF 是否损坏
