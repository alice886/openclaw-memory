### 关键区分

**青年刀马 vs 刀马17岁：**
- 青年刀马 = 成年后偏年轻版，c040 用这个
- 刀马17岁 = 真正的少年版，字面意思

**陈国武将 vs 陈国士兵：**
- 陈国武将 = 军官/武将，不是普通士兵
- 陈国士兵 = 普通士兵，c040 用这个

**线稿标记关键词（❌排除）：**
- RR = Rough Rendering = 线稿
- 线稿 / lineart / sketch / draft / 草图
- 轮廓 / outline
- 未上色 / uncolored

**✅ 可用标记：**
- 清稿 / 色稿 / 上色 / final / v2 / v3
- 三视图 / turnaround
