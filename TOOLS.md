# TOOLS.md - 本地笔记

Skill 定义了工具怎么用，这个文件放的是_我_的 specifics——只有我这里才有的东西。

## 这里放什么

比如：
- 摄像头名称和位置
- SSH 主机和别名
- 偏好的 TTS 语音
- 音箱/房间名称
- 设备昵称
- 任何环境相关的东西

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，移动触发

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 偏好的语音："Nova"（温暖，略带英式）
- 默认音箱：Kitchen HomePod
```

## 为什么分开

Skill 是共享的。你的配置是你的。分开放意味着更新 skill 不会丢失你的笔记，分享 skill 也不会泄露你的基础设施。

---

加上任何能帮你工作的东西。这是你的速查表。
